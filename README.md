# Doctor Survey Management App Backend 


## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npm run dev
   ```

You will be needing an sql server which i have used in my project 

Please feel free to use the mock data in mock.sql for testing
